# CAD Models — SolidWorks 2021

This directory contains all CAD-related files for the Solid Rocket Motor project.

## Files

| File | Description |
|------|-------------|
| `cad_3d_casing_model.png` | 3D isometric view of the motor casing |
| `cad_3d_nozzle_model.png` | 3D isometric view of the CD nozzle with flange |
| `cad_assembly_view.png` | Full SRM assembly — casing + nozzle mated |
| `cad_section_view.png` | Section/cutaway view of the assembly |
| `casing_2d_drawing.png` | Engineering 2D drawing of motor casing (with dimensions) |
| `nozzle_2d_drawing.png` | Engineering 2D drawing of nozzle (with section cuts) |
| `nozzle_cross_section.png` | CD nozzle profile showing convergent-throat-divergent zones |

## Design Approach

- **Software:** SolidWorks 2021 (Dassault Systèmes)
- **Method:** Bottom-up parametric modeling
- **Units:** mm (MMGS system)

## Key Dimensions

### Nozzle
- Convergent half-angle: 45°
- Divergent half-angle: 15°
- Throat diameter: ~6 mm
- Inlet diameter: 68 mm
- Exit diameter: 78 mm
- Flange OD: 128 mm
- Total length: 136.30 mm

### Casing
- Outer diameter: 128 mm
- Inner diameter: 88 mm
- Length: 250 mm
- Wall thickness: ~2 mm
- Flange bolt holes: 8× M6 at 88 mm PCD
