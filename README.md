# Rocket-motor-design-fabrication
End-to-end development of a solid rocket motor (SRM) , CAD modeling in SolidWorks 2021, CFD analysis of convergent-divergent nozzle using ANSYS Fluent (1500 iterations), and physical fabrication in mild steel via lathe &amp; TIG welding. Propellant: KNSU (KNO₃/Sucrose, 65:35 O/F ratio), tubular grain geometry, expected thrust 600–700 N
